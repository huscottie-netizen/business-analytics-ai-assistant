# Business Analytics AI Assistant

An MVP analytics assistant that combines:
- business KPI analysis
- data visualization
- AI-generated insights
- automated reporting

Built with:
- Streamlit
- Python
- pandas
- plotly
- Gemini API

---

# Features

## CSV Upload
Upload business sales data.

## KPI Dashboard
Automatically calculates:
- total sales
- growth rate
- coupon usage rate
- best-performing region

## AI Business Questions
Ask questions such as:
- Why did sales decrease?
- Which region performed best?
- What trends are visible?

## Automated Report Generation
Generate downloadable Markdown reports.
